# Lab 3

## Basic tool usage

The aim of this lab is to get familiar with the basics. By the end you'll be able to create, find, modify and delete resources within your cluster.

## Exercises  

- create a namespace
- start a pod
- add a service / ingress
- use a deployment
- delete a pods,services,deployments
